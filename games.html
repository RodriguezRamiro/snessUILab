Installed Cartridges

The following cartridges are currently registered within the SNES UI Lab console environment. Each cartridge operates as an independent module connected to the shared runtime engine.

Flight Simulator

Module: modules/games/flightSim.js

Status: Playable

Input: Keyboard

Version: 1.0

Core Systems

  • Self-contained game cartridge architecture
  • Directional movement and acceleration physics
  • Collision detection and crash handling
  • Obstacle spawning system
  • Difficulty scaling logic
  • Canvas-based rendering pipeline
  • Game Over state and restart cycle
